Overview

GeoJSON The purpose of this project was to work as a team, while simulating real world techincal roles, and the real world software development life cycles, to implement a fully funtional web application. This web application should access several online data sources through API's, extract relevant data from these sources, and present this data to the user in a way that is clear, simple and easy to analyze.
